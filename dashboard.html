File .env (nâng cao)

Sửa trực tiếp mọi cấu hình (token, key, giá token GEMINI_PRICE_IN_USD/OUT_USD...). Lưu xong bấm RESTART BOT để chắc ăn mọi giá trị mới. Bản cũ tự giữ ở .env.bak. CẨN THẬN: đổi BOT_DASH_TOKEN thì URL dashboard hiện tại hết hiệu lực.
